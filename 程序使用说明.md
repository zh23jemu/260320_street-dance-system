# 街舞信息共享平台程序使用说明

## 1. 项目简介

本项目为“街舞信息共享平台”毕业设计程序，采用前后端分离方式实现。

- 后端技术：Python、Django、SQLite
- 前端技术：React、Vite、JavaScript
- 主要功能：用户注册登录、活动发布与报名、视频展示与互动、聊天室交流、商城商品浏览与下单、个人中心数据查看

## 2. 交付内容

交付包中包含以下主要内容：

- Django 后端源码
- React 前端源码
- SQLite 数据库文件 `db.sqlite3`
- 一键启动脚本 `start_all.ps1`
- 单独启动脚本 `start_backend.ps1`、`start_frontend.ps1`
- 依赖说明文件 `requirements.txt`

## 3. 运行环境

建议在 Windows 环境下运行，主要依赖如下：

- Python 3.12 左右版本
- Node.js 18 及以上版本
- npm

## 4. 启动方式

### 4.1 推荐方式：一键启动

在项目根目录执行：

```powershell
.\start_all.ps1
```

执行后会分别打开前端与后端窗口。

### 4.2 分开启动

#### 启动后端

```powershell
.\start_backend.ps1
```

#### 启动前端

```powershell
.\start_frontend.ps1
```

## 5. 首次运行说明

如果本机尚未安装相关依赖，可按以下方式处理：

### 5.1 安装后端依赖

```powershell
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

如果没有本地虚拟环境，也可以先创建 `.venv` 后再安装依赖。

### 5.2 安装前端依赖

进入 `frontend` 目录后执行：

```powershell
npm install
```

## 8. 使用建议

建议按以下顺序进行演示：

1. 打开首页浏览平台内容
2. 登录演示账号
3. 进入活动页查看活动并报名
4. 进入视频页查看作品并评论或点赞
5. 进入社交页发送聊天室消息
6. 进入商城页加入购物车并创建订单
7. 在个人中心查看个人数据统计

## 9. 说明

- 本项目数据库已随程序一并提供，可直接用于演示
- 商城支付为模拟支付流程，不接入真实第三方支付
- 聊天室采用普通接口轮询方式，不是 WebSocket 实时通信
- 视频发布当前采用视频地址或文件路径方式演示
