# 街舞信息共享平台程序交付说明

## 1. 项目简介

本项目为“街舞信息共享平台”毕业设计程序，采用前后端分离方式实现。

- 后端技术：Python、Django、SQLite
- 前端技术：React、Vite、JavaScript
- 主要功能：用户注册登录、活动发布与报名、视频展示与互动、聊天室交流、商城商品浏览与下单、个人中心数据查看

## 2. 交付内容

当前交付包中包含以下内容：

- Django 后端源码
- React 前端源码
- SQLite 数据库文件 `db.sqlite3`
- 一键启动脚本 `start_all.ps1`
- 单独启动脚本 `start_backend.ps1`、`start_frontend.ps1`
- 依赖说明文件 `requirements.txt`

## 3. 系统功能概览

### 3.1 用户模块

- 用户注册、登录、退出
- 个人信息维护
- 关注、收藏
- 个人中心数据统计

### 3.2 活动模块

- 活动列表展示
- 发布活动
- 报名活动
- 活动签到
- 活动收藏
- 我的发布、我的报名

### 3.3 视频模块

- 视频作品展示
- 发布视频
- 点赞、收藏、评论
- 我的作品查看

### 3.4 社交模块

- 默认聊天室初始化
- 房间列表、房间详情
- 分类交流
- 登录后发送消息

### 3.5 商城模块

- 商品展示
- 加入购物车
- 创建订单
- 模拟支付
- 订单记录查看

## 4. 运行环境

建议在 Windows 环境下运行，主要依赖如下：

- Python 3.12 左右版本
- Node.js 18 及以上版本
- npm

## 5. 启动方式

### 5.1 推荐方式：一键启动

在项目根目录执行：

```powershell
.\start_all.ps1
```

执行后会分别打开前端与后端窗口。

### 5.2 分开启动

#### 启动后端

```powershell
.\start_backend.ps1
```

#### 启动前端

```powershell
.\start_frontend.ps1
```

## 6. 首次运行说明

如果本机尚未安装相关依赖，可按以下方式处理。

### 6.1 安装后端依赖

```powershell
.\.venv\Scripts\python.exe -m pip install -r requirements.txt
```

如果没有本地虚拟环境，也可以先创建 `.venv` 后再安装依赖。

### 6.2 安装前端依赖

进入 `frontend` 目录后执行：

```powershell
npm install
```

## 11. 技术说明

### 11.1 架构模式

本系统采用前后端分离的 Web 架构：

- 前端负责页面展示、交互逻辑和请求发送
- 后端负责业务逻辑处理、数据管理和接口返回
- 数据库负责平台结构化数据的存储与查询

### 11.2 技术选型

- 前端：JavaScript、HTML5、CSS3、React、Vite
- 后端：Python、Django
- 数据库：SQLite

当前阶段没有引入 Redis、消息队列和第三方支付服务，重点用于毕业设计原型演示与功能闭环展示。

## 12. 说明事项

- 本项目数据库已随程序一并提供，可直接用于演示。
- 商城支付为模拟支付流程，不接入真实第三方支付。
- 聊天室采用普通接口轮询方式，不是 WebSocket 实时通信。
- 视频发布当前采用视频地址或文件路径方式演示。
- 如需重新生成演示数据，可在后端环境下执行 `seed_demo_data` 相关命令。
